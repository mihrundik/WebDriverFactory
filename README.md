
# Инстукция по запуску тестов:

1. скачать на свой локальный компьютер проект
2. в пространстве проекта сформировать файл для установки переменных (пример во вложении - setenv.sh)
3. запустите терминал и перейдите в папку проекта
4. командой _source setenv.sh_ в терминале установить переменные
5. используя командную строку в терминале запустить тесты:
   - _mvn clean test_ - запуск с параметрами по-умолчанию
   - _mvn clean test -Dbrowser=safari_ - запуск с указанием браузера
   - _mvn clean test -Dbrowser=firefox -DfirefoxOptions='--kiosk,--incognito'_ - запуск с указанием браузера и опций через запятую
   - _mvn clean test -Dbrowser=chrom -DfirefoxOptions='--headless' -Dtest=FirstTest_ - запуск с указанием браузера, опций и названия теста

# Домашнее задание
## WebDriverFactory

### Цель:
Реализация автоматического теста с использованием Java + Selenium.

---

### Задание:

1. **Создание класса WebDriverFactory**:
   - Класс `WebDriverFactory` должен иметь статический метод `create()`, принимающий обязательный параметр `webDriverName` и необязательный параметр `options`.
   - Метод возвращает объект драйвера браузера, указанного в аргументе, с необходимыми настройками.

2. **Использование метода:**
   ```java
   WebDriver wd = WebDriverFactory.createNewDriver("chrome"); // Простое создание Chrome-драйвера  
   ```
   Или с дополнительными опциями:
   ```java
   FirefoxOptions options = new FirefoxOptions();  
   WebDriver wd = WebDriverFactory.createNewDriver("firefox", options); // Создание Firefox-драйвера с кастомными настройками  
   ```

3. **Написание скрипта для заполнения формы**:  
   Необходимо заполнить форму, включающую следующие поля:
   - Имя пользователя
   - E-mail
   - Пароль
   - Дата рождения
   - Уровень владения языком

4. **Логика проверки пароля**:  
   Добавьте механизм проверки соответствия введённого пароля и его повторения.

5. **Тестирование кнопки отправки**:  
   Проверьте, что форма отправляет данные корректно и выводит результат на экран.

6. **Паттерн PageObject**:  
   Используйте этот шаблон проектирования при создании тестов.
